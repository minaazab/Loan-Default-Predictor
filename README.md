# Loan-Default-Predictor
 Uses Data Science techniques, including the usage of pandas, Numpy, feature engineering, and visualization creation to predict the likelihood of loan defaulting while ensuring that the results are explainable and fair.


The dataset used in this project is too large to upload to github. Download the dataset from this Google Drive folder: https://drive.google.com/drive/folders/1kc7MQ0C2WPfDAhUf0jucCEp70o_P4p5y?usp=sharing


To run this project, I recommend trying to run it in Codebench, since I put a lot of time and effort in the formatting of the Jupyter Notebook. 

Alternatively, you could clone this project from github, download the datasets from Google Drive, place all the files in the same directory, and run pip install -r requirements.txt. This will ensure that all the dependencies are installed.
